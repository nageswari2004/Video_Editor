Process via Text Prompt

Use text commands for merging, overlaying, or other operations. Ensure you select the correct files below based on your command.

For 'merge videos', select additional videos here. For 'overlay at X,Y', select the overlay video/image here.
Prompt Help:
  • Merge Videos: merge_videos transition=fade duration=2
    • Requires: Main video + auxiliary videos. Transition types: fade, dissolve, cut
  • Apply Transitions: transition type=fade duration=1.5
    • Requires: Main video + auxiliary videos. Types: fade, dissolve, wipe, cut
  • Text Overlay: overlay type=text content="Hello" position=center duration=5
    • Works on main video only. Positions: center, top, bottom, top-left, etc.
  • Single Video Commands: trim start=10 end=30, speed factor=1.5, color_grade preset=cinematic
    • These work on the main video file only
How to use: Upload your main video, select additional videos in "Auxiliary Video Files" for merge/transition commands, then enter your command.
